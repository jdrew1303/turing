Turing emulator, using the Turing machine from the amazing book The Annotated Turing by Charles Petzold.

Hit 'g' to go, 's' to stop. Single step with 'n' / 'N' / 'm' (animation step to next state, step to next state, step to next cell).

The example here calculates root 2 in binary. All inlined javascript, no external dependencies. All state's in the browser DOM. Source is on GitHub.
